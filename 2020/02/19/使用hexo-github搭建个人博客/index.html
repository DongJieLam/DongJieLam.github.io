Created 2020-02-19| Updated 2020-05-30
Post View:

环境安装

这里是使用hexo和github来搭建个人博客。hexo是一款快速、简洁、强大的博客框架,可以帮助我们快速的搭建个人博客。由于hexo是基于Node.js的，所以在搭建之前要先安装node.js环境。我们再使用github来部署个人博客，就可以在公网访问到自己的博客。没有github的要先注册GitHub账户，并安装好git环境。这里主要是介绍搭建个人博客， 就不详细介绍如何安装git环境。

安装Node.js

  1. 到node官网下载node

  2. 打开安装程序，按照提示进行安装,安装位置根据自己的喜好选择就行，其他默认安装

  3. 安装完成后，win+r 输入cmd ,打开命令提示符，输入以下命令验证是否安装成功

安装git

  1. git官网下载，按提示默认安装就行
  2. 可以参考廖雪峰的git教程

安装hexo

  1. 打开cmd,输入命令安装hexo客户端，等待下载完成即可

  1. 验证是否安装成功。结果能出现版本号就安装成功

本地搭建博客

  1. 新建文件夹

在本地电脑找个地方新建一个文件夹，用来存放博客的文件。我在电脑的D盘的根目录下新建一个名为blog的文件夹。

  1. 命令行形式进入文件夹

    打开cmd, 输入d: 回车进入d盘。再输入 cd blog进入新建的文件夹。接下来都是用命令来创建博客的，所以要以命令行形式先进入文件夹

  2. 初始化博客

    输入命令 hexo init 进行初始化博客,在输入npm install

  3. 启动博客
